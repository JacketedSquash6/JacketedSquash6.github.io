Let's first get a better understanding of CSS

if we think of CSS in terms of selectors, properties, and values, then a typical CSS entry would look like this:

            selector {property: value;}

To style the above, we need to add two tags together. We call this CSS Combinators

What are Pseudo Classes?

again, if we think of CSS in terms of selectors, properties, and values, then a pseudo class is an add-on to a selector

            selector:pseudoClass {property: value;}

We use these for links!

If we want our link to look different, we can simply add it in my CSS as follows:

            a {color: red;}

But if we want each of our link states to look different then we will use a pseudo class:

  • :visited
  • :hover
  • :active
